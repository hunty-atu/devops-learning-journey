# Project: Build and Configure Server/VM

## Steps

1. **Create new VM in CRCTXVMCluster1**
2. **Name VM and select location CRCTXDATAC1**
3. **Place VM in CRCTXVMCluster1 compute resource**
4. **Select VxRail-Virtual-SAN-Datastore for storage**
5. **Choose ESXi 7.0 U2 and later version**
6. **Configure settings for Windows Server 2019 (64-bit)**
7. **Select network (DataVlanStack1-4) and CD/DVD drive (Datastore ISO File)**
8. **Use SW_DVD9_Win_Server_STD_CORE_2019 ISO image**
9. **Ensure VMXNET 3 adapter type for network**
10. **Review server details and launch web console**
11. **Follow steps to boot and set up Windows Server 2019**
12. **Install VMware Tools and complete setup**

14. **Edit settings to untick CD/DVD drive connection**
